# Minnesota Millionaires Lottery Club

**Official Dashboard:** [https://codycarlson2020-boop.github.io/Minnesota-Millionaires-Lottery-Club/](https://codycarlson2020-boop.github.io/Minnesota-Millionaires-Lottery-Club/)

## How it Works
This project runs automatically on GitHub Actions.
1.  **Scrapes** the latest MN Lottery results every night at 11 PM.
2.  **Checks** our club numbers against the winners.
3.  **Updates** the dashboard with our total winnings and spending.
